## 4. PDF Technical Requirements

### GoodNotes Compatibility Checklist

| Requirement | Specification | Priority |
|-------------|--------------|----------|
| PDF Version | 1.4–1.7 (1.7 recommended) | Critical |
| Color Space | sRGB | Critical |
| Font Embedding | Full embed or subset required | Critical |
| Hyperlinks | Internal PDF links (page destinations) | High |
| Bookmarks | PDF outline tree for navigation | High |
| Max File Size | <500 MB (marketplace), <100 MB recommended | Medium |
| Transparency | Supported in pages and annotations | Medium |
| Layers | Not used by GoodNotes | Low |
| Forms | AcroForms partially supported | Low |

### Hyperlink Implementation

```
Hyperlinks in GoodNotes MUST:
- Use /GoTo actions with page destinations (not named destinations)
- Use explicit /Rect coordinates for tap targets
- Have tap targets minimum 44×44 points (Apple HIG)
- Avoid /URI actions (external links stripped by some apps)
- Test on actual GoodNotes — some annotation subtypes render differently
```

### Font Embedding Best Practices

- **Always embed fonts** — GoodNotes doesn't have system font fallback for PDFs
- **Subset fonts** — Include only used glyphs to reduce file size (can save 70-90%)
- **Use TrueType (.ttf) or OpenType (.otf)** — Both supported
- **Verify licensing** — Google Fonts are all open-source (OFL/Apache)
- **Test special characters** — Date formats, currency symbols, emoji

### Color Space Management

```
- Author all templates in sRGB (IEC 61966-2-1)
- GoodNotes renders in sRGB on all devices
- P3 wide gamut on newer iPads but sRGB is safest
- AVOID CMYK — will display but colors may shift
- Dark backgrounds: note that Apple Pencil defaults to black strokes
- Recommended dark bg: #1E1E1E to #2D2D2D (not pure black)
```

## 8. Technology Stack Evaluation

### PDF Generation: Library Comparison

| Feature | pdf-lib | PDFKit | jsPDF | Puppeteer |
|---------|---------|--------|-------|-----------|
| **Hyperlinks** | Manual (low-level) | Built-in API | Basic | Via HTML |
| **Bookmarks** | Manual (low-level) | Built-in API | None | Via HTML |
| **Font Embedding** | ✅ Subsetting | ✅ Subsetting | ⚠️ Limited | Via HTML/CSS |
| **Custom Fonts** | TTF/OTF | TTF/OTF | TTF (limited) | Any web font |
| **Color Space** | RGB/CMYK | RGB | RGB | sRGB (browser) |
| **Vector Graphics** | SVG paths | Canvas-like API | Basic shapes | Full HTML/CSS |
| **File Size** | Small | Small | Medium | Large |
| **Complex Layouts** | Manual positioning | Manual positioning | Manual positioning | Full CSS layout |
| **Maintenance** | ⚠️ Stalled (2024) | ✅ Active | ✅ Active | ✅ Active |
| **TypeScript** | ✅ Native | ⚠️ Types exist | ✅ Native | ✅ Native |

### Our Architecture Decision

```
┌─────────────────────────────────────────────────────────┐
│                Template Generation Pipeline              │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  Complex Layouts (Planners, Journals):                  │
│  ┌─────────┐    ┌──────────┐    ┌──────────┐          │
│  │ HTML/CSS │ →  │ Puppeteer │ →  │   PDF    │          │
│  │ Template │    │  Render   │    │  Output  │          │
│  └─────────┘    └──────────┘    └──────────┘          │
│                                                         │
│  Hyperlinked Navigation (Planners with tabs):           │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐         │
│  │ Puppeteer │ →  │  pdf-lib  │ →  │  Final   │         │
│  │  (pages)  │    │ (links+  │    │   PDF    │         │
│  │          │    │  outlines)│    │          │         │
│  └──────────┘    └──────────┘    └──────────┘         │
│                                                         │
│  Stickers & Icons:                                      │
│  ┌──────────┐    ┌──────────┐    ┌──────────┐         │
│  │ SVG.js + │ →  │  Sharp   │ →  │   PNG    │         │
│  │  svgdom  │    │ (raster) │    │  Output  │         │
│  └──────────┘    └──────────┘    └──────────┘         │
│                                                         │
│  Simple Pages (Lined, Grid, Dot Grid):                  │
│  ┌──────────┐    ┌──────────┐                          │
│  │  PDFKit  │ →  │   PDF    │                          │
│  │ (direct) │    │  Output  │                          │
│  └──────────┘    └──────────┘                          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Why This Architecture?

1. **Puppeteer for complex layouts** — Full CSS Grid/Flexbox, Google Fonts, pixel-perfect typography, responsive design
2. **pdf-lib for post-processing** — Add hyperlinks, bookmarks, metadata to Puppeteer-generated PDFs
3. **PDFKit for simple pages** — Direct PDF construction is faster and smaller for grids/lines/dots
4. **SVG.js + Sharp for stickers** — Resolution-independent SVG → crisp PNG at any DPI
5. **Hybrid approach** — Each tool used where it excels, compensating for others' weaknesses
