# shared/

Assets shared across packs. Nothing pack-specific lives here.

- `themes/` — reusable CSS theme files (bold-tech, bubblegum, caffeine, candyland, claude, cyberpunk, doom64)
- `fonts/` — locally-hosted webfonts pulled by `scripts/download-fonts.ts`
- `base.css` — common reset + page/grid primitives (was `src/templates/html/_base.css`)
