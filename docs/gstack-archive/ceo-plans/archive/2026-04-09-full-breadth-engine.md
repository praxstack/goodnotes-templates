## Vision

### 10x Check
The 10x version is a community-driven template ecosystem with a web customizer, plugin system, AI-generated covers, locale support for 20+ languages, and automatic yearly releases. This plan builds the engine that makes all of that possible. The 10x version emerges naturally from a well-architected v1.0.

## Scope Decisions

| # | Proposal | Effort | Decision | Reasoning |
|---|----------|--------|----------|-----------|
| 1 | GitHub Releases asset bundle + download script | S | ACCEPTED | 95% of users won't run npm install. Downloads are the product. |
| 2 | Locale-aware date formatting (6 languages) | S | ACCEPTED | 15 min effort, opens entire international market. |
| 3 | "How to Import" instruction page in every PDF | S | ACCEPTED | Single biggest quality signal. Separates pro from amateur. |
| 4 | Custom theme JSON loader + example file | S | ACCEPTED | #1 feature request on every template repo. Zero-code customization. |

| 6 | Local web preview server | M | ACCEPTED | 10x dev velocity. Also generates README preview images. |

- GitHub Releases ZIP bundle with curated assets + auto-publish Action
