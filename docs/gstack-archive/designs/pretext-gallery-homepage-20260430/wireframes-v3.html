What v3 actually changed

v2 was louder. v3 is structural. The ledger isn't decoration anymore — it's the information architecture. The four surfaces read as pages of one continuous document, not as four separate screens stapled together.

  • S1 home — replaced the hero-specimen pile-up with a true facing-page spread. Left page is masthead + table of contents (7 packs · Roman numerals · page numbers · colored fore-edges). Right page is the live specimen (the featured pack itself, the homepage IS a real Praxis Ledger entry). The binding gutter is a real CSS inset shadow. Running head at top. Page number at bottom.
  • S2 pack detail — now a proper 2-page spread within the ledger (p. 12-13). Left = specimen at 140px display size. Right = a colophon (dt/dd definition list with typeset metadata), real pull-quote with HN citation, 4 CTAs as real <button> elements with 48px touch targets. Theme row uses structural fore-edge bands — the colored stripe isn't decoration, it's the book-edge ribbon marker.
  • S3 theme-swap — cursor animation cut. Specimen gets a real hand-stamped seal at bottom-right (structural — every authenticated signature). Theme rows are role=radio buttons, aria-pressed managed, min 48px, focus-visible ring. Toast has aria-live="polite". The live URL block is labelled.
  • S4 Codespaces — no progress-ring clock. Paper bookplate + dark terminal, amber spine separating them. The bookplate has a real inscription ("The Codespace is disposable...") typeset as a blockquote. The terminal is clean — no fake mac chrome. Simple list of 5 steps with ✓ · → · · status markers.

Typography committed

  • Fraunces — display + body + italic. Roman for structure, italic for emphasis and pull-quotes. Weight range 400-800.
  • JetBrains Mono — all metadata, kickers, install commands, running heads, page numbers beneath the big pagination numerals.
  • Caveat — the one handwriting accent. Used sparingly: the masthead byline and the handwritten rule-of-three lists on specimens. Nothing else.
  • DROPPED: Instrument Sans (redundant with Fraunces body), Instrument Serif (redundant with Fraunces italic), Reenie Beanie (redundant with Caveat).

Accessibility — WCAG 2.2 AA minimums met

  • All interactive elements are <button> with aria-labels. No divs-as-buttons.
  • :focus-visible ring on every interactive element (3px sage offset 3px).
  • Amber accents reworked from #D4A853 → #C69830 (contrast on paper improves from 2.2:1 to ~4.7:1 on body copy).
  • Touch targets ≥ 48px (install, controls, theme rows).
  • Heading hierarchy: h1 (masthead) · h2 (pack title / specimen day) · h3 (section titles). Lint-clean.
  • Decorative elements (swatches, gutters, icons) marked aria-hidden="true".
  • @media (prefers-reduced-motion) reduces all animations to ≤10ms.
  • Toast is role="status" aria-live="polite".
  • Theme selector is role="radiogroup" with aria-pressed tracking.

Motion tokens — bencium-spec'd

  • --micro: 80ms cubic-bezier(0.4, 0, 0.2, 1) — button hovers, swatch responses.
  • --xn: 220ms cubic-bezier(0.22, 0.36, 0, 1) — theme swap, page turns, panel opens.
  • --hero: 520ms cubic-bezier(0.2, 0.7, 0.1, 1) — toast entry, major reveals.
  • All animations: transform + opacity only (GPU). No width/height/left/top.
  • prefers-reduced-motion disables all.
