# Test Plan
Generated by /plan-eng-review on 2026-04-09
Branch: main
Repo: goodnotes-templates

## Affected Pages/Routes
- CLI `generate` command — primary generation flow
- CLI `preview` command — localhost Express server
- CLI `list` command — enumerate available templates/themes

## Key Interactions to Verify
- Theme loading: 8 built-in themes load with correct color/font values
- Custom theme: valid JSON loads, invalid JSON shows helpful error
- Dimension calculation: all 6 paper sizes × 2 orientations produce correct point values
- Locale: month/day names correct for en, es, fr, de, ja, ko
- PDFKit rendering: lined, grid, dot-grid, blank, isometric, music, calligraphy pages produce valid PDFs
- SVG sticker generation: all 14 types produce valid SVG markup
- Sharp rasterization: SVG → PNG at 300 DPI with transparent background
- Puppeteer rendering: HTML template → PDF with correct page dimensions
- pdf-lib post-processing: hyperlinks added with correct coordinates, bookmarks in outline tree
- Batch merge: 12 monthly chunks combine into single PDF preserving page order
- Manifest: JSON lists all files with correct metadata
- Instruction page: appears as first page, themed correctly

## Edge Cases
- Year=0 or year=9999 on CLI → validation error
- Missing font files → fallback to Inter with warning
- Custom theme JSON missing required fields → clear validation error
- Empty output directory → creates automatically
- --dry-run flag → lists files without generating
- Locale with no translation file → falls back to English

## Critical Paths
- Full planner generation: config → Puppeteer render → pdf-lib hyperlinks → output file
- Sticker generation: config → SVG.js → Sharp → transparent PNG
- npm run generate → all assets generated → manifest.json written
