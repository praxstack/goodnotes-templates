# Tactile Habit Tracker

A claymorphic A4 monthly habit tracker for GoodNotes. Part of the **Tactile Planner System**.

**Page sections**

| # | Block | Notes |
|---|-------|-------|
| 1 | Header | "Habit Tracker" + "Consistency is cozy." subheader, month/year washi pill |
| 2 | Habit of the Month | Spotlight card with claymorphic progress bar (12 / 31 days example) |
| 3 | Daily Tracking grid | 8 habits × 31 days of soft-clay cells; color-coded per habit (mint / rose / lavender / secondary) |
| 4 | Weekend strip | Faint washi tint behind Sat/Sun columns |
| 5 | Monthly Reflection | Three dotted writing lines in a clay card |
| 6 | Next Month Focus | Three dotted writing lines in a clay card |
| 7 | Footer | Encouragement quote + page label |

Form factor and palette match the rest of the Tactile Planner System (A4 portrait, cream paper, dusty rose / mint / lavender, Quicksand + Nunito Sans + Comfortaa + Be Vietnam Pro).

## Usage

Open `tactile-habits.html` → print to PDF at A4 portrait, no margins → import into GoodNotes. Day cells are rendered empty (no prefill); check off with the pen tool, or use GoodNotes' tap-to-toggle overlays.
