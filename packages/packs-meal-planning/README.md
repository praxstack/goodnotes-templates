# 🍽️ Meal Planning

Weekly meal grid — 7 days × breakfast/lunch/dinner. Sidebar for the grocery list.

| | |
|---|---|
| **Template** | [`meal-planning.html`](meal-planning.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One page per week |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-meal-planning/meal-planning.html \
  -o output/meal-planning.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
