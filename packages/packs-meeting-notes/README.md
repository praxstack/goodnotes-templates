# 🤝 Meeting Notes

Meeting capture — agenda · attendees · decisions · action items. Structured but not rigid.

| | |
|---|---|
| **Template** | [`meeting-notes.html`](meeting-notes.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One page per meeting |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-meeting-notes/meeting-notes.html \
  -o output/meeting-notes.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
