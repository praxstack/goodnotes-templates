# Shame + Baggage

**Kind:** data-capturing
**Captures:** what showed up (solid + dotted), where it landed in the body (dotted), a "noticed · didn't force" checkbox, one dotted line for any residue.
**Aggregation rule (month-end):** AI counts shame-spikes logged, notes body-location patterns (chest / stomach / throat), and surfaces repeat shame themes.
**Size class:** standard (600×600)
**Primary accent:** clay (heavy content, honest)

## Why this shape

Therapist-guided "notice, don't force" principle: name the shame, locate it, acknowledge it, don't need to fix it. The checkbox is the whole point — completion isn't the resolution, the noticing is.
