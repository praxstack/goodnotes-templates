# If-Then Plan

**Kind:** data-capturing
**Captures:** the cue (`IF` block, 1 solid + 1 dotted line) and the committed response (`THEN I'LL` block, 1 solid + 2 dotted).
**Aggregation rule (month-end):** AI counts triggers named and plans written. Flags repeated triggers (same cue three weeks running) for therapy-session agenda.
**Size class:** standard (600×600)
**Primary accent:** sage (growth / intention)

## Why this shape

Gollwitzer implementation-intention pattern — "if X, then I'll Y". Cue gets primary (solid) emphasis; response gets the slightly bigger writing area because the *response* is the thing that has to survive the moment.
