# Grateful for

**Kind:** data-capturing
**Captures:** one primary solid line + two optional dotted lines.
**Aggregation rule (month-end):** AI collects gratitude entries, surfaces recurring themes, tags people/places/things.
**Size class:** compact (400×600)
**Primary accent:** amber (warmth)

## Why this shape

Quick tap sticker — stick it anywhere, fill it in. One is enough; two is a bonus. Designed to be the cheapest possible emotional capture.

## Build

```bash
npx tsx scripts/build-stickers-remaining.ts
```
