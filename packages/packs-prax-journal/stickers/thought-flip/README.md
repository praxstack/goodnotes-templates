# Thought Flip

**Kind:** data-capturing
**Captures:** (1) the harsh thought the brain produced, (2) evidence it's true / not true, (3) the kinder-and-truer reframe.
**Aggregation rule (month-end):** AI counts how often each step has ink in it. Tally of cognitive-distortion patterns that keep reappearing; streak of days Prax actually finished the flip vs just wrote the thought.
**Size class:** expanded (800×600)
**Primary accent:** clay (confrontational without shouting)

## Why this shape

Classic Ramsay + Rostain CBT thought-record, compacted into three columns so the flow reads left-to-right. Column 1 (thought) is a primary solid line — one sentence, captured fast. Columns 2 (test it) and 3 (reframe) get three dotted lines each — space to actually work through it.

## Grammar

- 8px clay rail, clipped to squircle
- Kicker `§ THOUGHT FLIP` · hero `the flip` · subtitle `CBT · catch it, test it, rewrite it`
- 3 equal columns at ~227px each, primary + 3 secondary lines per column
- Whisper: `a thought is not a fact · give it one test`
