# Named Patterns

**Kind:** data-capturing
**Captures:** 5 named cognitive loops (doomscroll, catastrophize, prereq trap, rejection scanner, + 1 blank row for personal), each with 5 tick boxes.
**Aggregation rule (month-end):** AI sums ticks per pattern across the month and produces a frequency histogram. Dominant pattern becomes a retro talking point.
**Size class:** standard (600×600)
**Primary accent:** clay (pattern recognition, truth-telling)

## Why this shape

Prax's four default loops are pre-seeded so the user doesn't have to re-invent them each day. The 5th row stays blank for an emerging pattern. Five tick boxes per row capture "how many times in one day" without forcing a number field.

npx tsx scripts/build-stickers-remaining.ts
