# 📅 Weekly Planner

7-day layout — Monday through Sunday, with a left sidebar for weekly priorities and notes.

| | |
|---|---|
| **Template** | [`weekly-planner.html`](weekly-planner.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One week per sheet |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-weekly-planner/weekly-planner.html \
  -o output/weekly-planner.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
