# 🎨 Mood Tracker

Mood-over-time grid. Dot each day in its mood color. Watch patterns emerge across a month.

| | |
|---|---|
| **Template** | [`mood-tracker.html`](mood-tracker.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One month per sheet |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-mood-tracker/mood-tracker.html \
  -o output/mood-tracker.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
