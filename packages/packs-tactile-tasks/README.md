# Tactile Tasks & Notes

A claymorphic A4 tasks-and-notes planner for GoodNotes. Part of the **Tactile Planner System**.

**Page sections**

| # | Block | Notes |
|---|-------|-------|
| 1 | Header | "Tasks & Notes" + "One list at a time." subheader, today-pill |
| 2 | Top Priorities | 3 claymorphic priority cards (rose / mint / lavender) with title + supporting line + tag chip |
| 3 | To-Do List | 8 claymorphic round checkboxes with writing lines |
| 4 | Completed | Struck-through completed block with 3 pre-checked claymorphic ticks |
| 5 | Important | Tertiary-container callout with dashed rounded border |
| 6 | Brain Dump | Lined notes canvas with washi tape accent |
| 7 | Footer | Page label + helper quote |

Form factor and palette match the rest of the Tactile Planner System (A4 portrait, cream paper, dusty rose / mint / lavender, Quicksand + Nunito Sans + Comfortaa + Be Vietnam Pro).

## Usage

Open `tactile-tasks.html` → print to PDF at A4 portrait, no margins → import into GoodNotes. Checkboxes and rule lines are all static print surfaces; fill via pen or GoodNotes' tap-to-check stickers.
