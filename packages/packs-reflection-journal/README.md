# 🔍 Reflection Journal

End-of-day reflection sheet. What went well · what didn't · one thing to carry forward.

| | |
|---|---|
| **Template** | [`reflection-journal.html`](reflection-journal.html) |
| **Format** | A4 portrait · self-contained styles |
| **Use** | One sheet per evening. Three prompts, three answer zones. |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-reflection-journal/reflection-journal.html \
  -o output/reflection-journal.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
