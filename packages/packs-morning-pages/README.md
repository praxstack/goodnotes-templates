# ✍️ Morning Pages

Julia Cameron's [_Morning Pages_](https://juliacameronlive.com/basic-tools/morning-pages/) — three pages of longhand, stream of consciousness, written first thing in the morning. This template gives you one A4 sheet sized for that rhythm.

| | |
|---|---|
| **Template** | [`morning-pages.html`](morning-pages.html) |
| **Format** | A4 portrait · self-contained styles |
| **Use** | Draft 3 copies per day. No prompts. Unfiltered. |
| **Tap targets** | Free writing — no discrete targets. |

## Render

```bash
npx tsx src/cli/index.ts render \
  packages/packs-morning-pages/morning-pages.html \
  -o output/morning-pages.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
