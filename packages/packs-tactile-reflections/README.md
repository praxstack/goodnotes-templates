# Tactile Evening Reflections

A claymorphic A4 evening journal for GoodNotes. Part of the **Tactile Planner System**.

**Page sections**

| # | Block | Notes |
|---|-------|-------|
| 1 | Header | "Evening Reflections" display + "Take a deep breath. Today is done." handwritten sub |
| 2 | Date tape | Skeuomorphic washi date tape, tilted -1°, with event icon |
| 3 | Brain Dump | Large lined canvas, ~18 writing lines, soft-clay recessed container |
| 4 | Highlights of the day | Mint prompt card with 3 writing lines |
| 5 | Something I Learned | Lavender prompt card with 3 writing lines |
| 6 | For Tomorrow | Rose prompt card with 3 writing lines |
| 7 | Close Diary | Pill-shaped claymorphic sign-off button footer |

Form factor and palette match the rest of the Tactile Planner System (A4 portrait, cream paper, dusty rose / mint / lavender, Quicksand + Nunito Sans + Comfortaa + Be Vietnam Pro).

## Usage

Open `tactile-reflections.html` → print to PDF at A4 portrait, no margins → import into GoodNotes. All prompts render as dotted rule lines or bordered fields; fill with pen or type.
