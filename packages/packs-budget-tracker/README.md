# 💰 Budget Tracker

Income · expenses · savings across a month. Simple grid — not accounting software, just a weekly glance.

| | |
|---|---|
| **Template** | [`budget-tracker.html`](budget-tracker.html) |

```bash
npx tsx src/cli/index.ts render \

```
