# Tactile Daily Planner

A claymorphic, skeuomorphic A4 daily planner for GoodNotes. Part of the **Tactile Planner System** — a cozy, hand-made aesthetic that bridges physical scrapbooking and digital planning.

**Page sections**

| # | Block | Notes |
|---|-------|-------|
| 1 | Header | "Daily Planner" display, handwritten date line, weather badge |
| 2 | Today's Flow | 8 time-block rows (06:00 → 20:00), claymorphic recessed inputs |
| 3 | Mood Tracker | 5 puffy mood circles (very sad → very happy) |
| 4 | Hydration | 8 water-drop slots (4 full, 4 empty) |
| 5 | Movement | Step count field with running-figure badge |
| 6 | Daily Joy | Dashed rounded cloud for gratitude notes |
| 7 | Top Priorities | 3 claymorphic cards for high-importance tasks |
| 8 | Footer | Quote + binder spine |

**Form factor**

- **Paper:** A4 portrait (210 × 297 mm), single page
- **Gutter:** 14 mm left binder spine with 6 skeuomorphic rings
- **Palette:** Dusty rose (`#82505c`), Mint (`#466554`), Lavender (`#625a78`) on Cream (`#fbf9f4`)
- **Typography:** Quicksand (headlines) · Nunito Sans (body) · Comfortaa (handwritten accents) · Be Vietnam Pro (labels)
- **Print:** `@page size: A4 portrait; margin: 0;` · `-webkit-print-color-adjust: exact`

**Design inspiration**

Based on the **Digital Cozy** / **Tactile Planner System** design direction — claymorphism (inner highlight + inner shadow + soft drop shadow for extruded matte-plastic feel) layered over skeuomorphic paper cues (binder rings, washi tape, dotted rule lines).

## Usage

Open `tactile-daily-planner.html` in any browser → print to PDF at A4 portrait with margins set to none → import to GoodNotes. Writable fields are rendered as dotted rule lines, bordered boxes, or recessed clay containers; fill with the GoodNotes pen or type tool.
