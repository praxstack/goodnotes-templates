# 📝 Cornell Notes

Cornell method — two-column note page with a bottom summary zone. Left column for cues/questions, right column for notes, bottom for the summary.

| | |
|---|---|
| **Template** | [`cornell-notes.html`](cornell-notes.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One page per lecture/meeting |

## Render

```bash
npx tsx src/cli/index.ts render \
  packs/notes/cornell-notes/cornell-notes.html \
  -o output/cornell-notes.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
