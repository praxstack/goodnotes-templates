# Win Today

**Kind:** data-capturing
**Captures:** one checkbox + one solid primary line + one dotted secondary.
**Aggregation rule (month-end):** AI counts days with a "win today" logged vs blank; the "one thing" becomes the wins-jar feeder.
**Size class:** compact (400×600)
**Primary accent:** sage (single focus · anti-perfectionist)

## Why this shape

One check box. One writing line. "One is enough." The whole point is to refuse the multi-win cognitive load — pick one, name it, move on.
