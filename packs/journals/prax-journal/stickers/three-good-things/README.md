# Three Good Things

**Kind:** data-capturing
**Captures:** three lines — each line has a primary solid (the thing) + dotted secondary (optional why).
**Aggregation rule (month-end):** AI produces a "good things" list, tags entries by theme, flags days with zero good things as possible flags for therapy agenda.
**Size class:** standard (600×600)
**Primary accent:** amber (positive psych · warmth)

## Why this shape

Seligman's 3-good-things, rendered with space for a brief "why" under each. Numbers 01-03 in Fraunces set the editorial weight — feels like a magazine list of three best things, not a survey.

npx tsx scripts/build-stickers-remaining.ts
