# Craving Surf

**Kind:** data-capturing
**Captures:** the urge named (solid line), intensity on 1–5 scale (5 dots), rode-it-out yes/no, next-time plan (dotted line).
**Aggregation rule (month-end):** AI tallies craving frequency, mean intensity, ride-out success rate, and extracts next-time-plan themes.
**Size class:** standard (600×600)
**Primary accent:** lavender (pass-through, anti-shame)

## Why this shape

Marlatt urge-surfing worksheet — normalize the craving, rate its wave, note if it passed, commit to a different response next time. Quintile (1-5) scale feels right for urges; 10-point is too clinical for this.
