# 🐸 Eat the Frog

Brian Tracy's _Eat That Frog!_ — do the single most important (most unpleasant) task first each day. This sheet has one giant box for the frog, three smaller boxes for secondary tasks, plus a reflection line for what pushed back.

| | |
|---|---|
| **Template** | [`eat-the-frog.html`](eat-the-frog.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One page per day |

## Render

```bash
npx tsx src/cli/index.ts render \
  packs/worksheets/eat-the-frog/eat-the-frog.html \
  -o output/eat-the-frog.pdf
```
