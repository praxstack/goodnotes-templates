# 🥘 Recipe Card

Recipe capture — title · servings · ingredients · method · notes. Single card, fridge-ready.

| | |
|---|---|
| **Template** | [`recipe-card.html`](recipe-card.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One card per recipe |

## Render

```bash
npx tsx src/cli/index.ts render \
  packs/worksheets/recipe-card/recipe-card.html \
  -o output/recipe-card.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
