# 📆 Yearly Overview

12-month at-a-glance — a full year on one A4 sheet. Good for goal-setting, tracking long-running projects, or as the cover page of a multi-page planner.

| | |
|---|---|
| **Template** | [`yearly-overview.html`](yearly-overview.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One year per sheet |

## Render

```bash
npx tsx src/cli/index.ts render \
  packs/planners/yearly-overview/yearly-overview.html \
  -o output/yearly-overview.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
