# ✅ Habit Tracker

Monthly habit grid — one row per habit, one column per day, filled dots for completion.

| | |
|---|---|
| **Template** | [`habit-tracker.html`](habit-tracker.html) |
| **Format** | A4 portrait · self-contained styles |
| **Scope** | One month per sheet |

## Render

```bash
npx tsx src/cli/index.ts render \
  packs/trackers/habit-tracker/habit-tracker.html \
  -o output/habit-tracker.pdf
```

## Compatibility

GoodNotes 6 · Notability · Noteshelf · CollaNote.
